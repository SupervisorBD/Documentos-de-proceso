# Portal Corporativo Portable

## Qué hace
- Abre PDFs en un visor interno con botón de imprimir.
- Abre Excel, Word y formularios en una pestaña nueva.
- Mantiene los enlaces y los datos del archivo base.
- Permite mover elementos entre secciones y exportar el proyecto en JSON.

## Cómo usarlo en GitHub Pages
1. Sube `index.html`, `styles.css`, `app.js` y `data.js` al repositorio.
2. Sube la carpeta `docs/` con todos tus archivos.
3. En GitHub Pages, publica la rama principal desde la raíz del repo o desde `/docs` según tu estructura.
4. Asegúrate de que las rutas en los elementos usen `docs/nombre-archivo.ext` sin barra inicial.

## PDFs
- Al abrir un elemento PDF, se mostrará dentro del portal.
- Usa el botón **Imprimir** para sacar una copia.
- El botón **Abrir** permite verlo en una pestaña aparte.
